### Prerequisites
- Node.js 16+ 
- npm or yarn

### Setup

1. **Navigate to frontend directory**
   ```bash
   cd frontend
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure environment**
   ```bash
   cp .env.example .env.local
   # Edit with your API endpoint and configuration
   ```

4. **Start development server**
   ```bash
   npm run dev
   ```
